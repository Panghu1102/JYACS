# JYACS 设置 - 用户使用指南

## 简介
JYACS 设置现在有了独立的入口！你可以在游戏菜单中轻松访问所有 JYACS 相关的配置选项。

## 如何打开 JYACS 设置

### 方法 1: 通过游戏菜单（推荐）
1. 在游戏中按 **ESC** 键打开游戏菜单
2. 在菜单左侧找到 **"JYACS设置"** 按钮（在 "Return" 按钮下方）
3. 点击按钮即可打开 JYACS 设置界面

```
├── History

├── Discord Server

├── Official YouTube
├── Credits

└── JYACS设置       ← 新增！点击这里
```

## JYACS 设置界面说明

### 标签页
JYACS 设置界面包含两个标签页：

#### 1. 基础设置
包含日常使用的主要设置：

**API 配置**
- **API 密钥**: 你的 API 访问密钥（显示为星号保护隐私）
- **API 地址**: API 服务器地址
- **模型名称**: 使用的 AI 模型名称

**连接设置**
- **自动连接**: 游戏启动时自动连接 API
- **自动重连**: 连接断开时自动尝试重连
- **启用触发器**: 启用对话触发器功能

**功能设置**
- **启用情绪识别**: 启用 AI 情绪识别功能
- **回复时显示控制台**: AI 回复时显示调试控制台
- **启用 Mspire**: 启用 Mspire 功能
- **严格模式**: 启用严格的内容过滤

#### 2. 高级设置
包含 AI 模型的高级参数：

**超参数设置**
- **Temperature**: 控制回复的随机性（0.0-2.0）
  - 较低值：更确定、更一致的回复
  - 较高值：更有创意、更多样的回复
- **Top P**: 核采样参数（0.0-1.0）
- **Max Tokens**: 最大生成令牌数
- **Frequency Penalty**: 频率惩罚（减少重复）
- **Presence Penalty**: 存在惩罚（鼓励新话题）

**模式设置**
- **MF 激进模式**: Memory Fusion 激进模式
- **SFE 激进模式**: Semantic Feature Extraction 激进模式
- **ESC 激进模式**: Emotional State Capture 激进模式
- **NSFW 接受**: 是否接受 NSFW 内容

**其他设置**
- **随机种子**: 用于可重现的随机结果

## 常用操作

### 首次配置 JYACS
1. 打开 JYACS 设置
2. 在 "基础设置" 标签页中：
   - 点击 "API 密钥" 按钮，输入你的 API 密钥
   - 点击 "API 地址" 按钮，输入 API 服务器地址
   - 点击 "模型名称" 按钮，输入模型名称（如果需要）
3. 点击 "保存设置" 按钮
4. 点击 "验证配置" 按钮确认配置正确
5. 如果验证通过，JYACS 将自动连接（如果启用了自动连接）

### 修改 AI 回复风格
如果你想让 AI 的回复更有创意或更稳定：

1. 打开 JYACS 设置
2. 切换到 "高级设置" 标签页
3. 调整 **Temperature** 滑块：
   - 向左（降低）：回复更稳定、更可预测
   - 向右（提高）：回复更有创意、更多样
4. 点击 "保存设置"

### 启用/禁用功能
1. 打开 JYACS 设置
2. 在 "基础设置" 标签页中找到相应的功能开关
3. 点击开关切换状态（选中=启用，未选中=禁用）
4. 点击 "保存设置"

### 验证配置
如果你不确定配置是否正确：

1. 打开 JYACS 设置
2. 点击 "验证配置" 按钮
3. 查看提示信息：
   - 如果配置正确，会显示成功提示
   - 如果有问题，会显示具体的错误信息

## 按钮说明

### 保存设置
- **功能**: 保存所有修改并关闭设置界面
- **快捷键**: 无
- **提示**: 修改设置后记得点击此按钮保存

### 验证配置
- **功能**: 检查当前配置是否完整和正确
- **快捷键**: 无
- **提示**: 首次配置后建议验证一次

### 返回
- **功能**: 关闭设置界面，返回游戏菜单
- **快捷键**: ESC
- **提示**: 未保存的修改可能会丢失

## 常见问题

### Q: 我的设置保存了吗？
A: 只有点击 "保存设置" 按钮后，设置才会保存。如果直接点击 "返回" 或按 ESC，未保存的修改会丢失。

### Q: 为什么 API 密钥显示为星号？
A: 这是为了保护你的隐私。即使有人看到你的屏幕，也无法看到完整的 API 密钥。

### Q: 我应该使用什么 Temperature 值？
A: 推荐值：
- 日常对话：0.7-0.9
- 创意写作：1.0-1.2
- 稳定回复：0.5-0.7

### Q: 验证配置失败怎么办？
A: 检查以下项目：
1. API 密钥是否正确
2. API 地址是否正确
3. 网络连接是否正常
4. API 服务是否可用

### Q: 如何恢复默认设置？
A: 目前没有一键恢复功能，但你可以手动设置为以下推荐值：
- Temperature: 0.7
- Top P: 0.9
- Max Tokens: 2048
- Frequency Penalty: 0.0
- Presence Penalty: 0.0

### Q: 设置会在游戏重启后保留吗？
A: 是的，所有设置都会保存到 persistent 数据中，游戏重启后仍然有效。

如果你找到了完美的参数组合，可以记录下来，以便将来恢复。
