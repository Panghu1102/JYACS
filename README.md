## 什么是JYACS
*JYACS*是DDLC的MOD JustYuri的submod。主要通过内置的设置来配置API密钥，地址，模型名称，发送请求来实现AI对话功能。  
本项目与JustYuri进行了深度融合，但是由于目前最新汉化版所依赖的py版本为2.7，所以JYACS也按照了py2.7的语法来写，尽管有很多地方预留的py3的写法，在高版本的生肉里仍然可能有问题。  
如果后续有更新的汉化版本，我们会及时适配。
本项目与JustYuriDev没有任何关系。
